# Спецификация требований (SRS)

## 1. Назначение
Программный комплекс «PD Crypto Suite» предназначен для демонстрации
четырёх алгоритмов: шифр Виженера, SHA-384, метод хорд для решения нелинейных
уравнений и LSB-стеганография. Клиент-серверная архитектура поддерживает
авторизацию, роли и журналирование.

## 2. Стейкхолдеры
| Роль | Интерес |
|------|---------|
| Студент-разработчик | Реализовать функционал и сдать курсовую |
| Преподаватель | Проверить выполнение требований |
| Конечный пользователь (user) | Использовать криптофункции |
| Администратор (admin) | Управлять пользователями, читать логи |

## 3. Функциональные требования
| ID | Описание | Приоритет |
|----|----------|-----------|
| FR-01 | Регистрация пользователя по логину/паролю | M |
| FR-02 | Авторизация с проверкой SHA-384 хеша пароля | M |
| FR-03 | Шифрование/расшифрование текста по Виженеру | M |
| FR-04 | Вычисление SHA-384 произвольного текста | M |
| FR-05 | Решение уравнения f(x)=0 методом хорд | M |
| FR-06 | Внедрение/извлечение сообщения в PNG (LSB) | M |
| FR-07 | Список пользователей доступен админу | S |
| FR-08 | Блокировка/разблокировка пользователя админом | S |
| FR-09 | Просмотр журнала событий админом | S |
| FR-10 | Несколько одновременных клиентов | M |

## 4. Нефункциональные требования
| ID | Описание |
|----|----------|
| NFR-01 | C++17, Qt 6, кросс-платформенность (Linux/macOS/Windows) |
| NFR-02 | Сборка через CMake ≥ 3.16 |
| NFR-03 | Сервер запускается в Docker-контейнере |
| NFR-04 | Покрытие алгоритмов unit-тестами (QtTest) |
| NFR-05 | Документация Doxygen |
| NFR-06 | Время отклика на команду ≤ 200 мс при 10 клиентах |
| NFR-07 | Пароли хранятся только как SHA-384 хеш |

## 5. Роли и права
| Действие | user | admin |
|----------|:----:|:-----:|
| Регистрация / Вход | ✔ | ✔ |
| Виженер / SHA / Хорда / Стег | ✔ | ✔ |
| Список пользователей | ✘ | ✔ |
| Блокировка пользователя | ✘ | ✔ |
| Просмотр логов | ✘ | ✔ |

## 6. Интерфейсы
- TCP/JSON line-protocol (см. [`server/include/protocol.h`](../server/include/protocol.h:1)).
- GUI: Qt Widgets, вкладки по функциям + админ-панель.

## 7. Ограничения
- Стеганография ограничена ёмкостью ⌊(N-32)/8⌋ байт для N байт пиксельного буфера.
- Парсер выражений поддерживает `+ - * / ^`, `sin/cos/exp/log/sqrt`, скобки, переменную `x`.
