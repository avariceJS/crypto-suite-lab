# Метод хорд (regula falsi)

Численный метод нахождения корня уравнения `f(x) = 0` на отрезке `[a, b]`,
где `f(a)·f(b) < 0` (функция меняет знак).

## Итерация
```
x = a − f(a) · (b − a) / (f(b) − f(a))
```
Затем заменяется тот из концов отрезка, где знак функции совпадает с `f(x)`.

## Условия применимости
1. `f` непрерывна на `[a, b]`.
2. `f(a)·f(b) < 0`.

## Критерий остановки
`|f(x)| < ε` или достигнут лимит итераций (по умолчанию 1000).

## Парсер выражений
Поддерживается ввод формулы строкой:
- бинарные: `+ − * / ^`
- унарные: `−`, `+`
- функции: `sin`, `cos`, `exp`, `log`, `sqrt`
- скобки `()`, переменная `x`

См. [`pd::math::evalExpr`](../../core/include/chord.h:1).
