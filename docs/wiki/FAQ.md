# FAQ

**Q: Какие учётные данные по умолчанию?**
A: `admin / admin`. Меняйте при первом входе (планируется в `feature/admin-pwd`).

**Q: Не подключается клиент.**
A: Проверьте, что сервер запущен и порт `5555` открыт; в Docker-режиме
`docker ps` покажет контейнер `pd_server`.

**Q: Картинка после стеганографии «битая» при сохранении в JPEG.**
A: LSB-метод требует формата без потерь — сохраняйте только в **PNG/BMP**.

**Q: Метод хорд возвращает `converged=false`.**
A: Проверьте, что `f(a)·f(b) < 0` и функция гладкая на отрезке.

**Q: Где смотреть логи?**
A: Админ-вкладка → «Журнал событий» (команда `LOG_LIST`).
